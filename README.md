Timeless, MERN Stack eCommerce Project

Timeless is a full-stack eCommerce web application currently in development. It is built using the MERN stack, which includes MongoDB, Express.js, React.js, and Node.js.

This project serves as a long-term learning effort with a strong emphasis on backend development, database design, and real-world eCommerce processes.

Project Overview

Timeless aims to offer a complete eCommerce experience. Users can explore products, apply filters and sorting, select product variants like size, and manage a shopping cart.

Users can place orders by providing delivery details and choosing between Cash on Delivery or online payment methods. The application is being designed for scalability and a clean structure.

Planned User Features

Users can browse and explore products.  
They can filter and sort product listings.  
They can select product variants like size.  
They can add products to the cart and manage cart items.  
Users can place orders by providing a delivery address.  
They will have payment options that include Cash on Delivery and online payments using Stripe and Razorpay.

Planned Admin Features

An admin dashboard will be available.  
Admins can upload and manage products.  
They can delete products.  
Admins can view all products listed in the store.  
They can manage users and orders.

Backend and APIs

The backend is being developed with Node.js and Express.js.  
RESTful APIs are being created for products, users, and orders.  
MongoDB stores product, user, and order data.  
Mongoose is used for schema design and validation.

Technology Stack

Frontend technologies include React.js, HTML, CSS, and JavaScript.  
Backend technologies include Node.js and Express.js.  
MongoDB is used as the database with Mongoose.  
Online payments are planned using Stripe and Razorpay.

Project Structure

The project has a frontend folder for client-side code.  
There is a backend folder for server-side code.  
The root directory contains the README file and gitignore file.

Project Status

The project is currently in development.  
The basic project structure has been established.  
Frontend and backend folders have been created.  
Core backend logic and APIs are actively being developed.

This is a long-term project that will continue to evolve and improve.
